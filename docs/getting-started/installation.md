# Installation

## Requirements

- Python 3.11 or higher
- A browser with WebDriver BiDi support (Chrome, Edge, or Firefox)

## Install

```bash
pip install bidiwave
```

## Development install

```bash
git clone https://github.com/MathiasPaulenko/bidiwave.git
cd bidiwave
pip install -e ".[dev]"
```

## Verify

```python
import bidiwave
print(bidiwave.__version__)
```
